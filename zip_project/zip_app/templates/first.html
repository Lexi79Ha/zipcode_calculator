How To Use Zip-Calculator

Zip-Calculator is a tool that can allow users to find the distance in miles between any two zipcodes in the United States. To use this tool please input a valid 5 digit zipcode in each box and then press calculate. Zip-Calculator is designed to only accept US zipcodes. It will not accept any letters or special characters.
